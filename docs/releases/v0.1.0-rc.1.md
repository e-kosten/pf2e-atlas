# v0.1.0-rc.1

## Summary

The first PF2e Atlas release candidate. It publishes the `atlas` CLI as
downloadable GitHub release binaries for macOS, Linux, and Windows on x64 and
ARM64.

This RC is intended to validate the install, setup, and packaging flow before the
first stable release.

Highlights:

- Search Pathfinder Second Edition records locally with text search, semantic
  search support, structured filters, lookup, name resolution, and graph context.
- Run `atlas setup` to fetch or update PF2E source data, prepare local runtime
  data, and verify readiness.
- Install the first-party local-agent skill with `atlas agent skills install`.
- Install prebuilt binaries without requiring Rust or `cargo`.

## Install Notes

Install this release candidate from the pinned release URL. RCs are not marked as
the latest release.

macOS or Linux:

```bash
curl --proto '=https' --tlsv1.2 -LsSf \
  https://github.com/e-kosten/pf2e-atlas/releases/download/v0.1.0-rc.1/atlas-installer.sh | sh
```

Windows PowerShell:

```powershell
irm https://github.com/e-kosten/pf2e-atlas/releases/download/v0.1.0-rc.1/atlas-installer.ps1 | iex
```

After installing:

```bash
atlas --version
atlas setup
atlas setup --check
```

The installer only installs the `atlas` executable. Runtime data, including the
PF2E source checkout, embedding model cache, and local search artifact, is
managed by `atlas setup`.

## Known Issues

- Binaries are not yet code signed or notarized. macOS and Windows may show
  operating-system warnings when running the CLI for the first time.
- This is a release candidate. Installer behavior, release asset layout, and
  setup diagnostics are the main things being validated before `v0.1.0`.
