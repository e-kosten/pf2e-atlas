# v0.1.0-rc.2

## Summary
Reruns the initial release after fixing release packaging on GitHub Actions,
including Linux ARM64 runner compatibility and Windows x64 installer validation.

This release candidate publishes Linux x64, Linux ARM64, macOS Apple Silicon,
and Windows x64 binaries. macOS Intel and Windows ARM64 binaries are deferred
until the native ONNX Runtime packaging strategy supports them cleanly.
